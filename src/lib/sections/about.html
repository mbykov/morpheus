Morpheus for Eastern Languages:

- offline web application. Works offline, synchronizes when online

- works everywhere on desktop

- based on CouchDB + Electron.js, so distributed, multiuser, open-source, cross-platform and scalable

- instantly displays several paragraphs of text

- has recursive segmentation of a long word (segmenting of segments)

- user can install / connect / remove dictionaries on the fly, English, German, Russian, etc reliable open-source dicts

- only installed dictionaries are synchronized

- user can add / edit dictionary's articles (login required)

- includes "Early Chinese Buddhist translations" dictionaries prepared by P.Hauer

- Cf. "About ECBT dictionaries" Section
